Setup Guide · Windows

PDF Restriction Remover

One-time setup, then double-click to run anytime. Processes your entire folder at once and never touches your originals.

1

Check if Python is Already Installed

  1. 1
    Press Windows key + R on your keyboard
  2. 2
    Type cmd and press Enter — a black window opens
  3. 3
    Type this exactly and press Enter:
python --version

If you see something like "Python 3.x.x" — great, skip to Step 2.
If you see an error — Python isn't installed yet. Go to python.org, click the big yellow Download button, run the installer. Important: check the box that says "Add Python to PATH" before clicking Install.

2

Install the Required Library (One Time Only)

In the same black Command Prompt window, type this and press Enter:

pip install pypdf

You'll see some text downloading and installing. Wait for it to finish — usually under 30 seconds.

You only ever need to do this once. The library stays installed on your computer permanently.

3

Put the Script Somewhere Easy to Find

  1. 1
    Save the file pdf_restriction_remover.py somewhere easy — your Desktop or Documents folder works well
  2. 2
    Also save this to your own GitHub repository so you never lose it
4

Run It

  1. 1
    Double-click the pdf_restriction_remover.py file to open it
  2. 2
    A window appears asking for your PDF folder path
  3. 3
    Open File Explorer, find your bank statements folder, and drag the folder into the window — the path fills in automatically
  4. 4
    Press Enter and watch it process every file
  5. 5
    When done, look inside your statements folder for a new subfolder called "unlocked" — your renamed files are in there

If double-clicking doesn't open a window, right-click the file instead → Open with → Python.

5

What You Get

  1. ✓
    Every PDF processed and saved as originalname_pwd_rmvd.pdf
  2. ✓
    All files in a clean "unlocked" subfolder
  3. ✓
    Your originals untouched — exactly as they were
  4. ✓
    Unlocked files are now fully searchable and ready to redact in Adobe
