# Bank Statement Tools

Small scripts for working with bank statement PDFs:

- **pdf_restriction_remover.py** — removes permission restrictions from PDF bank statements so they can be searched, copied, and redacted.
- **rbc_statement_categorizer_v2.py** — extracts transactions from RBC statement PDFs and builds a categorized Excel summary.
- **bank-categorizer-dark.jsx** — a sample React UI that shows categorized transactions with totals by category (sample data is entirely fictional).
- **pdf-remover-instructions.html** — a plain-language setup guide for the PDF restriction remover script.
